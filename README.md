# 🐉 Dragon New

A modern news platform built with Next.js where users can explore categorized news, authenticate securely, and experience dynamic routing in action.

---

## 🔗 Live link & Repository

🌐 Live Site: https://m-47-dragon-news.vercel.app 
📂 GitHub Repo: https://github.com/mahdihasanprogrammer/dragon-news.git  

---

## ✨ Key Features

- 📰 Dynamic category-based news loading  
- 🔁 Automatic redirect to default category on homepage  
- 🔐 Authentication (Email/Password + Google & GitHub)  
- 👤 Conditional Navbar (Login / Profile + Logout)  
- 📢 Breaking news marquee section  
- 📂 responsive layout  
- 🎯 Active category highlighting  
- 🔒 Protected routes (About page)  
- 🔄 Dynamic routing with real-time category updates
